# QuestIonAble File Manager 0.4.0

Release `0.4.0` is the first build under the QuestIonAble File Manager name.
It changes the public repository, Pages path, Windows display name, .NET
projects and namespaces, canonical release assets, CLI executable, and
environment-variable prefix.

Update continuity is preserved:

- the signed Windows package identity remains
  `MesmerPrism.MetaQuestFileManager`;
- former `0.3.x` App Installer and asset URLs resolve through byte-identical
  compatibility aliases;
- portable releases retain the former CLI executable as a deprecated alias;
- former environment-variable names remain deprecated fallbacks.

The rename does not change ADB scope, permission boundaries, Rusty Kiosk
authority, file-operation behavior, or the project’s independent,
non-affiliated status.
