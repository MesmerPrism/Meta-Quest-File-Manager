Move files and APKs between Windows and Quest.

Browse ADB-accessible headset storage, install your own APKs, and export sideloaded single-APK apps with a SHA-256 checksum.

What it does

Browse accessible storage

Start at /sdcard, open folders, pull selected files, and explicitly push files to the current path.

Install your APKs

Choose reinstall, downgrade, runtime-permission, and test-package options before installation.

Retrieve sideloaded apps

Export an installed app only when Android reports one APK path. Split packages are refused rather than copied incompletely.

Requirements

  • Windows 10 or later
  • Android SDK Platform Tools
  • Quest Developer Mode
  • USB debugging approval in-headset

Important limits

APK export does not include app data, saves, downloaded assets, OBB files, licenses, or entitlements. The app cannot access paths Android denies to the ADB shell user.
